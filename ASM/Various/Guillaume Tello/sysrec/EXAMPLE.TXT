En exemple,le fichier SYSREC.INF fourni va lancer le petit programme

As an example, thefile SYSREC.INF provided is set to run the little tool
TSTSCR_F.PRG and to record every Xbios or VDI call (see line r=VXO). Upon
exit, you can read the file SYSREC.TXT to see the system call list that
has been recorded with their calling address (option "O").
