                         MidiShare packages

---------------------------------------------------------------------

Install/AUTO :  
		MidShare.Prg
		MidShare.Inf

Applications:
	Programs:
		MSDISK.PRG - direct to disk MidiShare sequencer
		MSECHO.PRG - MidiShare echo
		MSFLOW.PRG - Port/Canal routing
		MSNET.PRG  - MidiShare Patchbay
		MSPLAY.PRG - MidiShare player
		MSSYNTH.PRG - ST internal synthesizer
		with the corresponding ressource files.
	Desk Accessories :
		MSDISK.ACC   - direct to disk MidiShare sequencer
		MSECHO.ACC   - MidiShare echo
		MSFLOW.ACC   - Port/Canal routing
		MSNET.ACC    - MidiShare Patchbay
		MSPLAY.ACC   - MidiShare player
		MSSCREEN.ACC - Display midi messages handler
		MSSPACE.ACC  - MidiShare memory manager
		MSSYNTH.ACC  - ST internal synthesizer

Develop:
	C:
		LATTICE_C:	header file and libraries for Lattice C
		MWC:		header file and libraries for Mark Williams C
		TURBO_C:	header file and libraries for Turbo C
		SKELETON:	Midi software skeleton
		SRC:		source code for example applications
		TUTORIAL:	a tutorial for MidiShare application development.
	GFABasic: 
		MicroSeq.Lst - example of sequenceur application
		MidShare.Lst - interface file of MidiShare
		MSEcho.Lst   - example of echo application
		Skeleton.Lst - application skeleton for GFA Basic
		Lisez.Moi    - documentation en francais
		Read.Me      - english documentation
		with corresponding ressource files.

Documentation/Inline doc :
		MidiHelp.Acc  : MidiShare Help Desk Accessory
		MidiHelp.Prg  : MidiShare Help Program
		MidShare.Hlp  : MidiHelp Help ressource
		MidShare.Idx  : MidiHelp Help ressource
		Help.inf      : MidiHelp Help ressource
