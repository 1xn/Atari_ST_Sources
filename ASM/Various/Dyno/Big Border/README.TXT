 /   SYNTAX TERROR BIG BORDER REVISITED BY DYNO | SOMMARHACK 2017   \
 '-------------------------------------------------------------------'
  :                                                                 :
  : Back in 1990, Delta Force released the legendary Syntax Terror  :
  : megademo for the Atari ST. The Big Border demo screen was made  :
  : by Chaos, Inc. of Delta Force and it's main characteristics is  :
  : the tiny 96*54 pixel screen size.                               :
  :                                                                 :
  : Rules for the compo :                                           :
  :   * The demo is limited to a 96*54 area, centered on the screen :
  :   * No rasterbars outside of the 96*54 area                     :
  :   * Must work on 512k RAM                                       :
  :   * Must work on Atari ST                                       :
  :                                                                 :
  : Here is my tribute (working on Atari 520 ST), as a contribution :
  : to the Sommarhack 2017 demoparty that took place in             :
  : Grådö (Sweden) on july 8th 2017.                                :
  :                                                                 :
  : The commented source code is attached if anyone is interested.  :
  :                                                                 :
