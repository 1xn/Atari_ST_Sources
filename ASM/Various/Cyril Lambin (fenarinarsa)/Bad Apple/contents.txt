This archive contains everything needed to generate the BadApple!! conversion for Atari STE by fenarinarsa

BASTGenerator/
  C# data file generator
  Project created with Visual Studio 2015 Community

ba.s
  68000 source of the data file player
  Compile with vasm 68k mot http://sun.hasenbraten.de/vasm/

badapple.mp4
  the original video posted on nico nico, if needed

badapple_st12m_16.wav
  12kHz mono audio

badapple_st50_16.wav
  50kHz stereo audio

original/
  original PNG sequence converted from the mp4

resized/lowres/
  320x200 PNG sequence to be converted in PI1 by BASTGenerator

resized/highres/
  640x400 PNG sequence to be converted in PI3 by BASTGenerator
