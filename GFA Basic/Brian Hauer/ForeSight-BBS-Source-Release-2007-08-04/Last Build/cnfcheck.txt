CNF-Check.  A support utility for use with Foresight BBS
software to help verify the integrity of your FS.CNF
configuration file before running Foresight.  This utility
will help you detect problems with FS.CNF before any
troubles result from them.  For Foresight versions 0.47
Wide Beta and 0.48 Beta.
