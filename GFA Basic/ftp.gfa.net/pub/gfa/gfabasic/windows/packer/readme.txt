This directory contains several 3rd party DLLs to expand
files from different archive formats.
LHA and ZIP files can be created, too.
Please pay attention upon the different licence rules.

--------------------------------------------------------

Dieses Verzeichnis enhält verschiedene DLLS von Dritt-
Herstellern, mit denen Sie Dateien aus verschiedenen
Archiv-Formaten extrahieren können.
LHA- und ZIP-Archive können auch erstellt werden.
Bitte beachten Sie die verschiedenen Lizenz-Bedingungen
der einzelnen Hersteller.
