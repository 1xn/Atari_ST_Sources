This directory contains many complete, powerful applications,
entirely written in GFA-BASIC for Windows, that come with
source code.

In addition to the applications itselves you get tons of 
source codes you can learn from and integrate into your
own applications.

A special thank to all the authors who have released their
source code.

--------------------------------------------------------------

Dieses Verzeichnis enthält viele komplette, mit vielfältigen
Funktionen ausgestattete Applikationen, die allesamt in GFA-
BASIC für Windows geschrieben wurden - und alle im Source-
Code vorliegen.

Zusätzlich zu den Applikationen selbst erhalten Sie eine
große Menge an Sourcecode, von dem Sie lernen und den Sie in
Ihre eigenen Applikationen integrieren können.

Ein spezielles Dankeschön geht an alle Autoren, die ihren
Sourcecode freigegeben haben.
