
CPXMAKER is designed to be installed in the Devpac 3 Tools Menu and called
with its (.MPC) control file in the top window. See DEFAULT.MCP for details
of the control file format.

Use the following settings in the Tools menu entry:

Command Line:   As shown
Directory:              Tool's
Save Files:             Yes
Path:           D:\DEVPAC3\TOOLS\CPXMAKER.TTP (or wherever it is)
Command:                %?
Pause on return:        No
Report all errors:      No
Run as TOS:             Yes
Run as GEM:             No
