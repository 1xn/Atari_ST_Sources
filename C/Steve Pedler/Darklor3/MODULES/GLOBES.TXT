Module GLOBES.DMO
(Control file: GLOBES.DKL)

1. This module draws wireframe globes, either spheres or ellipses, in
a variety of sizes.

2. Flag 1 chooses between spheres or ellipses.

3. Variables 1 and 2 set the minimum and maximum sizes for the globe
radius. The actual radius set will be chosen randomly from between 
these 2 values.
