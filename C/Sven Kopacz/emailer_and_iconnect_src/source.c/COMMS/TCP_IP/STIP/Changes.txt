Änderungen in MacSTIP 0.9c:---------------------------- Neue Funktion GetMyIPAddr zur Abfrage der eigenen IP-Addresse und Netzmaske- Neue Funktion TCP_GetStatus zur Abfrage des Status einer TCP-Verbindung- Die Funktion GetDNRIPAddr ist zwar implementiert, liefert aber vorerst keinen  gültigen Wert- In der Funktion TCP_Send könenn jetzt die Flags urgent und push angegeben werden- Die Function TCP_Receive liefert jetzt das Flag urgent
