                            Superlib  V1.0
                            --------------
                            
   Die Diskette enthaelt mehrere Libraries im LASER C (TM) Format.
   Die Libraries enthalten einige (!) nuetzliche Funktionen, die
   zur Unterstuetzung der Original-Libraries benutzt werden koennen.
   Eine Beschreibung aller enthaltenen Funktionen ist ebenfalls auf
   der Diskette.

   In dem Ordner examples ist eine Beispiel-Source und ein makefile
   zum Erzeugen des executes dafuer.

   Die Funktionen sind in Hinblick auf Portabilitaet mit *NIX-Umgebungen
   entwickelt worden. Aus diesem Grunde wird auch eine Termcap-Datei,
   die die Funktionalitaet des Bildschirmes (Monochrome) wiederspiegelt,
   unterstuetzt und von verschiedenen Funktionen benutzt.


   Wem die Funktionen zusagen und wer gerne die neueste Version haette,
   auch fuer andere C-Development-Systems, der kann eine formatierte
   Diskette (3.5 oder 5.25) mit einem frankierten und adressierten
   Rueckumschlag und eine Bemuehungsanerkennung von DM 20.- an unten-
   stehende Adresse schicken.
   Ruecksendung garantiert innerhalb einer Woche.
