Juergen Buchmueller's Text To BASic converter

Usage is "text2bas program.bas" or "text2bas progam.txt output.vz"
In the first case it will create a file "program.vz" while in the
latter case you can specify a different output filename.
Just try "text2bas test.bas" and load test.vz into VZEm then.

Notes by Matthias Jaap:
The Atari version isn't a TTP program but it understands the same
commandline parameters. If you start cgentoken/vz200token without any
parameters, a file selector will pop up. All error messages are 
displayed in alert boxes.

New in Atari version 1.1: Text2Bas uses the CFLib, i.e. you need CFLib to
compile the sources. The file selector is displayed in a window now.

cgentoken: Generates basic files for the Colour Genie 
vz200token: Generates basic files for the VZ200
