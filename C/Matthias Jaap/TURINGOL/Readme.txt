				TURINGOL

This is Peter Gammie's Turingol interpreter, with some typos fixed and slightly
improved packaging.  The only functional change is that the default action is
-r rather than -o -- to dump the computation history, not the tuples in the
intermediate representation

Adapted for the Retrocomputing Museum by Eric S. Raymond, Feb 2000..

For instructions on how to build and use Turingol, see turingol.html.
