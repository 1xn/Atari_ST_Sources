NAME
     iso-time - _IDT-Cookie auf ISO-kompatibles Format setzen

SYNOPSIS
     \auto\iso-time.prg

BESCHREIBUNG
     Dieses Programm setzt den _IDT-Cookie auf eine ISO-8601-kompatible 
     Darstellung (YY-MM-DD).

     Falls der Cookiejar noch nicht existiert oder bereits voll ist, 
     wird das automatisch miterledigt.

EXIT STATUS
     0    Alles in Ordnung
     >0   Es ist ein Fehler aufgetreten
