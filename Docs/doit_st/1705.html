23.5 DSL


DSL mit dem Atari


T-DSL mit dem Atari, geht das?


Ja, das geht.
Wenn der Atari z.B über das Romport-Interface von Elmar Hilgart verfügt kann, bei korrekter Installation von STinG 1.26, der Atari an leicht T-DSL teilnehmen. Mit IConnect so wie einigen anderen Atari TCP/IP Sockets ist das nicht möglich da diese keine Netzwerkfähigkeiten besitzen.

Gibt es einen DSL PPPoE-Treiber für Atari?

Nein, einen solchen Treiber gibt es nicht.

T-DSL kann man mit dem Atari machen wenn man einen DSL-Hardware Router wie z.B den DrayTek Vigor 2000, Vigor 2200 besitzt. Solche Router findet man bei z.B bei TRK

Diese Router haben das PPPoE-Protokoll in ihrer Software eingebaut. Ein per Ethernet hier angeschlossener Atari benötigt dann keinen eigenen PPPoE-Treiber.

Besitzt man Computer anderer Plattformen, z.B einen Macintosh, kann hier auch ein Software DSL-Router, wie z.B IPNetRouter oder SurfDoubler, installiert werden von dem dann auch der im gleichen Subnetz angeschlossene Atari profitieren kann. Der Nachteil dabei ist das der Internetzugang ausschliesslich erfolgen kann wenn auch der Computer auf dem der Souftware-Router installiert ist, eingeschaltet ist.

Wie man ein Ethernet-Interface und STinG installiert finden sie an anderer Stelle hier im Archiv ausführlich beschrieben.
