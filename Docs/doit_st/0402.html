4.2 TT internal Connectors


Some urgend internal TT connectors



J100
Pin Bezeichnung/Spannung
1 = PG (PowerGood)
2 = +5 Volt
3 = +12 Volt
4 = -12 Volt (seriell!)
5 = GND
6 = GND
7 = GND
8 = GND
9 = -5 Volt 10 = +5 Volt Mainboard
11 = +5 Volt Mainboard
12 = +5 Volt Mainboard

 
J901
TT LAN port
 
J902
serial port "A"
 
J903
serial port "B"
 
J602
"I/O" pins.. ? I'm not realy sure if it is correct!!
2 + 4 = GND
1 = IO1 (Pin 26 U807)
3 = IO0 (Pin 25 UZ807)

 
J102
AUX Fan Pin
2+3 = GND
1 = +12

 
J103
PowerON Pin
2 = GND
1 = R105 (120R)

 
J803
internal Speaker Pin
1 = C854 (47uF) (Speaker)
2+3 = GND

 
W601
ChipEnable /EpromSize
Pin1 = A14
Pin2 = A16 any Eprom Socket
Pin3 = GND

 
W602
ChipEnable /EpromSize
Pin1 = Vcc (+5V)
Pin2 = A15 any Eprom Socket
Pin3 = A17 (AdressBus)

 
W603
ChipEnable /EpromSize
Pin1 = GND
Pin2 = CE of any Eprom Socket
Pin3 = A18 (AdressBus)

 
J203
Memory Select
