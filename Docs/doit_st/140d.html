20.13 Exposure Lichtsteuerung


Exposure steuert Lichtanlagen


Licht steuern mit Exposure

Exposure ist eine Software für Atari Computer die es ermöglicht Lichtdimmer vom Computer aus zu steuern. Alle Steuerungsausgaben erfolgen über die RS232 Schnittstelle mit einer Geschwindigkeit von 9600 Bps.

Es sind zwei Schaltinterfaces verfügbar, in Abhängigkeit davon was der Benutzer damit steuern möchte.

PMX Interface (Pulsar Multiplex).
Die Firma Pulsar ist ein grösserer Hersteller von Beleuchtungsartikeln.

Die Interfaces Pulsar Demux 18 ch oder Pulsar Demux 36 ch sind ideal geeignet um Beleuchtungen zu steuern. Das Demux Interface konvertiert die digitalen Steuersignale des Computers in ein analoges Signal von 0-10V DC welches von vielen Laststeuerbaugruppen direkt verarbeitet werden kann.

Ein weiteres Interface ist das DMX:1990 (Digital Multiplex).
Das DMX 1990 entspricht dem Digitalstandard. Die Durand Interstellar DMX Box konvertiert das RS232 Schnittstellensignal auf ein für die DMX Box verständliches 250Kbps Signal. Die DMX Box verarbeitet maximal 120 Steuerkanäle.

Eine neuere Softwareversion von Exposure befindet sich in Entwicklung.

Die neue Version wird einige wichtige Bugfixes beinhalten und wird neue Effekte wie, Lichtscanner, Stroboskopeffekte zusätzlich unterstützen.

Der Autor gibt allerdings zu bedenken das ein 8 Mhz ST für diese neue Version unter Umständen nicht mehr ausreichen wird. Die neue Version wird weiterhin als Atari-Version erscheinen, ebenfalls wird es dabei auch eine Windows-Version geben.
