20.10 Fischertechnik


Fischertechnik AtariST Interface


Für alle Fischertechnik-Computing-Interessierte habe ich eine kleine FAQ zusammengestellt. Weitere Fragen koennen direkt bei den fischerwerken unter der email fischertechnik-service@fischerwerke.de gestellt werden.

Frage:
Werden noch Computing-Interfaces mit Software fuer den Anschluss an ATARI-Computer vertrieben?

Antwort:

Das "alte" Universal-Interface (30520) laesst sich mit dem Adapter (32721) an den Atari anschliessen.

Frage:
Waren die alten Interfaces fuer C64/ATARI/AMIGA/PC baugleich und unterschieden sie sich nur beim Anschlussadapter für den jeweiligen Computertyp bzw. in der Computer-Software?

Antwort:
Das kommt darauf an, wie dieses Interface beschriftet ist. Steht dort "Fischertechnik Interface", so ist es mit dem entsprechenden Adapter universell einsetzbar. Falls auf dem Interface z.B. "C64 Interface" vermerkt ist, ist es speziell.

Frage:
Können die neuen Interfaces auch an ATARI-Computer angeschlossen werden und mit der alten Software angesteuert werden?

Antwort:
Nein, das ist nicht möglich.

Frage:
Kann ich bei Ihnen noch die letzte aktuelle Software fuer Atari bekommen?

Antwort:
Nun, wir haben noch die Software fuer Atari ST

Frage:
Was kostet die Software?

Antwort:
Nichts. Der Service ist umsonst
