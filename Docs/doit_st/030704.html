3.7.4 STBook Stromversorgung


STBook Stromversorgung




In diesem Artikel finden Sie die Anschlußbelegung des 120 poligen STBook-Expansionsport.

Netzteil

 
Laut Netzteil wird bereitgestellt:

 
DC Output: 16V 1,25A
Battery Charge Output (DC) 12V 1,20A

 
Im Leerlauf liefert das Netzteil folgende Spannungen:

 
(0)-(3): 17V
(0)-(2): 0V
(0)-(1): 16,8V

 
Die Leuchte für den Batterie-Ladevorgang am Netzteil leuchtet nur dann auf, wenn ein Akkupack eingesteckt ist, welches "Ladebedarf" hat.

 
Akku

 
Pinbelegung

 
Die Buchse sieht etwa so aus (Außen-Draufsicht)

  
   .---.
  | ° * |
  |  *  |
   \___/

Ringsum der Schirm (0). Links oben eine kleine Buchse (° -> 1), rechts daneben (* -> 2) und fast zentral in der Mitte (* -> 3) jeweils eine größere Buchse.
