Haben Sie schon einen Blick in das deutsche Revolution Magazin geworfen?
Revolution ist ein deutsches Atari Magazin das alle 2-3 Monate auf einer 650 MB CD-ROM erscheint. Das Medium kann auf jedem Atari mit CD-ROM Laufwerk oder einem Mac/PC mit MagiC PC oder MagiCMac gelesen werden. Das Magazin erscheint in einer Monochromen- und einer Farbversion als.htmlL damit es möglichst viele Anwender erreicht. HTML Dokumente können Sie auf jedem Atari mit CAB, dem Atari Crystal Internet Browser, lesen. CAB ist FreeWare. CAB 2.7 ist kommerziell und kann bei Applikation Systems Heidelberg bezogen werden.

  • CAB-Internet Browser
